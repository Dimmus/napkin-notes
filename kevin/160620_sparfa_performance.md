## Model

```
O( NC*(NL + NQ + iters*NL*NQ) ) == O(NC*NQ*NL*iters)
```
```
double NC --> x 1.4
double NQ --> x 1.4
double NL --> x 1.85
```

```
ONE RUN:
NL*2^8   --> 1_000_000 --> x 134
NQ*2^7.3 -->    64_000 --> x  12
NC*2^5.3 -->     6_400 --> x   6

269 * 134 * 12 * 6 = 2_600_000 sec = 720 hours = 30 days 
```

```
PER-SUBJECT:
NL*2^4.6 --> 100000  --> x 17
NQ*2^4   -->   6400  --> x  4
NC*2^2   -->    640  --> x  2

269 * 17 * 4 * 2 = 36000 sec = 10 hours / subject
```

```
SAMPLING PER-SUBJECT:
NL=4000 --> 25 samps --> x 1
NQ= 640 --> 10 samps --> x 1.25
NC=  80 -->  8 samps --> / 1.4

total samps = 25 * 10 * 8 = 2000
time per samp = 269 * 1 * 1.25 / 1.4 = 240 sec
2000 * 240 = 480000 sec = 133 hours = 5 days
on 100 cores, 1.5 hours
```
